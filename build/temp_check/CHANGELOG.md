## [1.0.0] - 2026-03-16

### 当前功能

**安装与激活**

- 扩展安装后随嘉立创 EDA 启动自动激活，无需手动触发。
- 在首页、原理图、PCB 顶部菜单栏注入「MCP Bridge」菜单，提供「连接设置」和「关于」入口。

**连接设置**

- 支持配置桥接服务地址，地址格式实时校验，非法格式拒绝保存并提示错误。
- 设置页实时显示桥接连接状态和 WebSocket 连接状态。
- 配置变更后自动重新连接，无需重启 EDA。

**桥接连接**

- 激活后自动连接到 MCP 服务端，断线后自动重连。
- 支持多个 EDA 实例同时接入，由服务端自动裁决哪个实例执行任务。
- 当前页面不是原理图或 PCB 编辑页时，连接自动暂停。

**EDA 工具支持**

- 支持 API 检索、API 调用、当前上下文获取三个 AI 工具，AI 通过这三个工具操控嘉立创 EDA。
